Mandelbrot-Zoom

A tool to explore the Mandelbrot set and it's infinite complexity.

Note that the webapp is being hosted on a free hoasting service, meaning that the computational resources are limited. For a snappier experience consider running the webapp locally:

Github Repository

Iteration count: Determines how quickly or slowly a point in the complex plain converges. So the deeper you are the higher the iteration count needs to be

Pixel density: Determines the size of the image that is produced

Resize: Ctrl + +/- , Ctrl + Scroll
